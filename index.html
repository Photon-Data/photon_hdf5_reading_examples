Photon-HDF5 Reading Examples

This repository contains code examples on reading Photon-HDF5 files in multiple languages.

Each folder contain examples for a specific language (Python, MATLAB or LabVIEW). For python, we provide both notebooks and scripts.

To execute the examples please download this repository archive (Download ZIP) and extract it: it will contain an empty data folder. Next, download the example Photon-HDF5 files and unzip them in the empty data folder.

For more info on reading Photon-HDF5 files refer to the official Photon-HDF5 documentation, in particular:

  • Reading Photon-HDF5 files
